# Quickstart Guide

## 5-Minute Setup Guide

### Installation Steps
1. **Clone the Repository**
   ```bash
   git clone https://github.com/ankushthakurajtk/form-automation-system.git
   cd form-automation-system
   ```

2. **Install Dependencies**
   Make sure you have [Node.js](https://nodejs.org/) installed. Then run:
   ```bash
   npm install
   ```

### Running Locally
1. **Start the Application**
   ```bash
   npm start
   ```
   The application should now be running on [http://localhost:3000](http://localhost:3000).

### First Automation Example
1. **Create a New Automation**
   Navigate to the automations section of the application and click on 'Create New Automation'.

2. **Fill in the Details**
   Enter the required form fields and click 'Save'.

3. **Run the Automation**
   Select the automation from the list and click 'Run'. 
   You should see it executing the steps you specified.

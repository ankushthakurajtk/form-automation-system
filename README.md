# Form Automation System

## Project Overview
The Form Automation System is designed to streamline the process of creating, managing, and analyzing forms for various applications. This system automates the tedious tasks involved in form handling, allowing users to focus on the important aspects of data collection and processing.

## Features
- **Dynamic Form Creation**: Create forms on-the-fly with various input types.
- **Real-Time Data Validation**: Ensure data integrity with validations at the time of input.
- **User Management**: Manage users with different access levels.
- **Analytics Dashboard**: Visualize and analyze form responses efficiently.
- **Export Options**: Export collected data in multiple formats (CSV, PDF, etc.)

## Tech Stack
- **Frontend**: HTML, CSS, JavaScript, React
- **Backend**: Node.js, Express
- **Database**: MongoDB
- **Deployment**: Docker, AWS

## Quick Start Guide
1. **Clone the repository**:
   ```bash
   git clone https://github.com/ankushthakurajtk/form-automation-system.git
   cd form-automation-system
   ```
2. **Install dependencies**:
   ```bash
   npm install
   ```
3. **Run the application**:
   ```bash
   npm start
   ```
4. **Access the application**:
   Open your browser and go to `http://localhost:3000`
